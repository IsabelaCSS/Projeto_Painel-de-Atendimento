Painel De Atendimento Hospitalar – Descrição do Projeto  

Integrantes: Isabela de Oliveira, Laura Sthefanny, Fernanda Matias, Letícia Pereira e Thais Gracielli  

O painel de atendimento hospitalar foi desenvolvido com a finalidade de simular o atendimento em hospitais e clínicas. A aplicação mostra em uma única tela, a senha atual, o guichê de atendimento, as senhas anteriores e os números dos consultórios.  

O atendimento é organizado por senhas exibidas no telão da sala de espera, sendo classificadas como atendimento comum ou prioritário. Ao serem chamados, os pacientes são direcionados a sua consulta, sendo exibido sua senha e a sala que deverá ir. Também será mostrado as últimas senhas chamadas, para que os pacientes não se percam.  

O sistema apresenta também uma área com o objetivo de gerir os atendimentos, onde é possível adicionar pacientes e atualizar o status de cada um. Além disso, foi incluído uma seção de mídia, para que as pessoas possam assistir enquanto esperam ser chamadas.  

O projeto é constituído por três partes principais:  

HTML, que dá a estrutura para a página e organiza os componentes.  

CSS, que define a configuração visual do painel, como cores, tamanhos, posicionamento e responsividade  

JavaScript, que controla as funcionalidades dinâmicas, como mover itens para o histórico e manipular a lista de atendimentos.  

De modo geral, o projeto mostra como combinar essas três linguagens para criar um sistema funcional, visando informações importantes e tendo como propósito ser claro e objetivo para quem está aguardando, para que assim, todos possam ser atendidos da melhor maneira.
